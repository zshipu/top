这适用于 Windows 用户。我使用cmd 和VS 代码终端来编写我的命令。请确保您在计算机上安装了 Python 、pip（Python的包管理器 - 就像npm是 Javascript）和Visual Studio 代码，并在 VS 代码中启用了 Python 扩展码。我假设您熟悉 Python 和命令行/如何编写基本命令。

我们要实现什么：

安装虚拟环境并处理它

第一步：导航到 Django 项目及其所有依赖项将驻留并安装 的文件夹。virtual environment

虚拟环境（也称为 Python 环境）是 Python 环境，因此安装到其中的 Python 解释器、库和脚本与其他虚拟环境中的脚本隔离。Python 虚拟环境的核心目的是venv``````to create an isolated environment for Python projects. This means that each project can have its own dependencies, regardless of what dependencies every other project has.
